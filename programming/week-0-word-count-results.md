# Week 0 Word-Count Diagnostic

Program: [word-count.py](word-count.py)

Input: [sample-input.txt](sample-input.txt)

Run from the repository root:

```powershell
python programming/word-count.py programming/sample-input.txt
```

Expected and observed output:

```text
17
```

Counting rule: a word is a Unicode letter/digit/underscore sequence that may contain an apostrophe. Punctuation and line breaks do not count as words.
